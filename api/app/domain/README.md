# Core

## General
This folder represents the core logic of the application, aka the domain logic layer.  The API layer is a wrapper around the core logic layer.  The _crud layer keeps basic operations separate from the core logic layer.

In non-monolithic applications the core logic layer is split from the api layer repository.  Having the api layer and the logic layer in separate folders will allow us the flexibility of refactoring towards that if needed (currently not needed) and also allows for better separation of concerns and easier testing.# Core
